## Run it on your machine

1. Clone the project

   ```sh
   git clone https://github.com/Arindam2209/Crypto-Verse.git
   ```

1. Install the dependencies

   ```sh
   cd Crypto-Verse
   npm install
   ```

3. Start the development server

   ```sh
   npm start
   ```
